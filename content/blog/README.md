# Blog Posts

Add new blog posts as `.md` (Markdown) files in this directory.

## Format

Each post should have YAML frontmatter at the top:

```markdown
---
title: "Your Post Title"
date: "2025-09-15"
---

# Your Post Title

Content goes here...
```

## Rules
- **Filename**: Use kebab-case, e.g., `my-new-post.md`
- **Frontmatter**: Must include `title` and `date` fields
- **Date format**: `YYYY-MM-DD`
- **Content**: Standard Markdown — headings, lists, code blocks, images, links all work
